About this project

This project is a simple simulation to the concept of messaging/blogging that we experience in social media
The purpose of tis project is to present the social media concept in a low-level by breaking it down to allow
learner/starter developers to be able to digist those concepts ...

This project is an open-source where it can be accessed through Github. Suggestion for improvment and modification are always welcome.
